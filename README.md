# Immersion Reader — 沉浸式阅读创作平台

一个基于浏览器的沉浸式阅读创作平台。创作者可以为文本段落搭配背景图片与音乐、编排时间轴，生成 `.ims` 文件；读者加载文件后即可获得自动播放、音画同步的沉浸式阅读体验。

## 🎯 功能概览

### 编辑器（Editor）
- 输入作品元数据（标题、作者、创作者、简介）
- 自由添加 / 删除文本段落
- 为每个段落选择背景图片和背景音乐
- 精确设置每段的起始时间和持续时长
- 选择段落切换效果（无动画 / 淡入 / 交叉淡入淡出）
- 保存作品为 `.ims` 文件，或打开已有文件编辑

### 播放器（Player）
- 加载 `.ims` 文件，自动解析时间轴
- 背景图片全屏显示，支持 CSS 淡入淡出切换
- 文字居中覆盖在背景上（白色文字 + 半透明黑色背景框）
- 背景音乐随段落自动切换
- 播放 / 暂停 / 手动翻页（上一段 / 下一段）
- 音量调节滑块
- 进度存档与读档（10 个槽位，数据保存在浏览器本地）
- 键盘快捷键支持

## 🛠 技术栈

| 层级 | 技术 |
|------|------|
| 后端框架 | Python Flask（≤ 100 行） |
| 前端技术 | 纯 HTML + CSS + Vanilla JavaScript（零框架） |
| 数据存储 | 文件系统（JSON），不使用数据库 |
| 构建工具 | 无（保存即生效，无需 npm / webpack） |
| 浏览器支持 | Chrome 90+ / Edge 90+ |

> **为什么不用前端框架？** 本项目 JS 逻辑约 300 行，原生 `<audio>`、`fetch()`、CSS `transition` 完全覆盖需求。零构建工具，团队成员可在 30 分钟内理解全部前端代码。

## 📦 安装与运行

### 环境要求
- Python 3.8+
- 现代浏览器（Chrome 90+ / Edge 90+）

### 安装步骤

```bash
# 1. 克隆或下载项目到本地
cd web_reader

# 2. 安装依赖（仅 Flask）
pip install flask

# 3. 启动后端服务
python app.py

# 4. 打开浏览器访问
# http://localhost:5000
```

浏览器打开后，默认进入播放器页面。通过导航栏可在编辑器、播放器和帮助页面之间切换。

## 📖 使用说明

### 创作流程（编辑器）
1. 打开 **编辑器** 页面（`/editor`）
2. 在左侧面板填写标题、作者、创作者等元数据
3. 点击「+ 添加段落」创建段落卡片
4. 在每个段落中填入文字、选择背景图和背景音乐、设置时间和切换效果
5. 点击「💾 保存」将作品保存为 `.ims` 文件

### 播放流程（播放器）
1. 打开 **播放器** 页面（`/player`）
2. 从下拉菜单选择一部作品，或通过 URL 参数自动加载（如 `/player?file=sample.ims`）
3. 点击「▶ 播放」开始自动播放
4. 使用空格键暂停 / 继续，方向键手动翻页
5. 使用存档功能保存进度，方便下次继续

### 快捷键

| 按键 | 功能 |
|------|------|
| 空格键 | 播放 / 暂停 |
| → 方向键 | 跳转到下一段 |
| ← 方向键 | 回到上一段 |

### 目录说明

| 目录 / 文件 | 作用 |
|------------|------|
| `app.py` | Flask 后端，提供页面路由和数据 API（文件列表、加载、保存、存档） |
| `templates/` | 前端 HTML 模板，包含编辑器、播放器和帮助三个页面 |
| `static/` | 静态资源，包含全局 CSS 样式表 |
| `data/` | 数据存储目录，存放 `.ims` 作品文件及图片、音频资源 |
| `savedata/` | 播放进度存档目录，每个槽位对应一个 JSON 文件 |
| `docs/` | 项目文档，包含完整的开发规格说明书 |

## 📄 `.ims` 文件格式

`.ims` 是 Immersion Reader 定义的作品文件格式，本质为 JSON。核心结构如下：

```json
{
  "version": "1.0",
  "metadata": {
    "title": "作品标题",
    "author": "原文作者",
    "creator": "创作者",
    "description": "作品简介"
  },
  "assets": {
    "images": { "别名": "相对路径" },
    "audios": { "别名": "相对路径" }
  },
  "timeline": [
    {
      "segment_id": 1,
      "text": "段落文字",
      "start_time": 0.0,
      "duration": 8.0,
      "background": "图片别名",
      "background_music": "音频别名",
      "transition_effect": "cross_fade"
    }
  ]
}
```

> 详细规范见 [docs/规格说明书_v1.0.md](docs/规格说明书_v1.0.md) 第 3 节。
