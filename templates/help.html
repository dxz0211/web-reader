Immersion Reader v2.0 — 沉浸式阅读创作平台

📖 概述

Immersion Reader 是一个沉浸式阅读创作平台，支持创作者制作带有背景图片、背景音乐和过渡效果的交互式阅读作品，读者可以在播放器中以手动或自动方式逐句阅读。

✏️ 编辑器使用说明

  1. 填写作品信息：在左侧面板填写作品标题、原文作者、创作者和作品简介。
  2. 添加段落：点击「＋ 添加段落」按钮创建新的段落。
  3. 添加句子：每个段落内可添加多个句子，播放时每点击一次显示一句。
  4. 设置入口段落：勾选「🔔 入口」复选框的段落可以设置背景图、背景音乐和切换效果。未勾选的段落会自动继承上一入口的设置。
  5. 上传资源：通过右侧面板上传图片和音频文件，上传后可在段落的下拉菜单中选用。
  6. 保存作品：点击「💾 保存」按钮或按 Ctrl+S 保存为 .ims 文件。

▶️ 播放器使用说明

  1. 加载作品：在上方下拉菜单中选择一个 .ims 文件，点击「加载」按钮。
  2. 手动阅读：加载后显示第一段第一句，点击「▶ 下一句」或按 空格 逐句推进。
  3. 自动播放：点击「🔄 自动」开启自动模式，每句话显示3秒后自动切换到下一句。
  4. 自动+手动混合：自动模式中仍可点击「下一句」手动提前切换，计时器会自动重置。
  5. 章节跳转：使用「◀ 上段」「下段 ▶」按钮或键盘 ← → 键切换段落。
  6. 回到开头：点击「⏮ 开头」按钮返回作品第一段。
  7. 音量调节：使用底部的音量滑块调节背景音乐音量。

💾 存档与读档

播放器支持10个存档槽位，方便保存和恢复阅读进度：

  1. 点击「💾 存档」按钮，选择槽位保存当前进度。
  2. 点击「📂 读档」按钮，选择存档槽位恢复之前的阅读位置。
  3. 存档会记录当前所在的段落索引，下次加载自动跳转到该位置。

📖 回想模式

点击「📖 回想」按钮可以查看作品所有段落的完整文字内容，方便回顾上下文。当前段落会以高亮标记。

📂 IMS 文件格式

.ims 是 Immersion Reader 的作品文件格式，使用 JSON 存储。其结构包含：

  • metadata：作品元数据（标题、作者、创作者、简介）
  • assets：资源映射（图片和音频文件的别名与路径）
  • timeline：段落时间轴，每个段落包含：
    • entry_point：是否为入口段落（决定是否切换背景和音乐）
    • background / background_music：背景图片和音乐的别名
    • transition_effect：切换效果（cross_fade / fade_in / none）
    • sentences：句子数组（播放时逐句显示）

旧版 .ims 文件（使用 text 字段而非 sentences 数组）能被自动兼容加载。
